Configuración requerida

Para usar Bontxo Agent necesitas conectar tu base de datos de Supabase.

  1. Ve a supabase.com
  2. Crea un proyecto (o usa el existente)
  3. Ve a Settings → API
  4. Copia la URL y la Key anónima
  5. Pégalas en el recuadro de abajo
